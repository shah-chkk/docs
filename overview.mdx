# Chkk Upgrade Intelligence

Chkk Upgrade Intelligence is an MCP-compatible framework that powers AI-driven Kubernetes upgrades. It generates pre-verified upgrade plans, exact diffs, and workflows directly in your IDE or AI agent.

## Key Features
- **Safe Upgrades**: Pre-validate steps across clusters and add-ons.
- **IDE Integration**: Works with Cursor, Claude Code, Windsurf, and VS Code.
- **Automated Workflows**: Fetch diffs, stage patches, and monitor health checks.

This documentation covers setup, getting started, and integration guides.
